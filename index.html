Despre noi

Rogentos Linux Group este grupul de initiativa pentru dezvoltare a sistemelor bazate pe Gentoo si a tehnologiilor adiacente si aferente

Rogentos Linux Group isi propune sa dezvolte sistemele de operare bazate pe Gentoo si Sabayon GNU/Linux numite Kogaion si Argent Linux.

Kogaion reprezinta varianta Desktop orientata catre utilizatorul cotidian de GNU/Linux. Este varianta preferabila pentru incepatori.

Argent reprezinta varianta orientata catre modul Server. Este inca in dezvoltare, nepublicata. Este utilizata momentan de dezvoltatori.
